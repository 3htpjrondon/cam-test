Was Cookbook
============

Installs and configures WebSphere Application Server and WebSphere Application Server Network Deployment

Requirements
------------

### Platform:

* Redhat (>= 7.0)
* Ubuntu (>= 16.0.4)

### Cookbooks:

* ibm_cloud_utils
* im
* linux
